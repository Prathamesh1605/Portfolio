# Prathamesh Shingan Portfolio

Modern, recruiter-ready portfolio powered by React, Vite, and Tailwind CSS. Content is fully driven by `src/data/resume.js`.

## Tech Stack
- React 18 + Vite
- Tailwind CSS
- React Three Fiber + Drei (3D visuals)

## Getting Started
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
npm run preview
```

## Update Resume Content
Edit `src/data/resume.js`. The UI will adapt to missing fields gracefully.

## Assets
- Resume PDF: `public/Prathamesh-Shingan.pdf`

## Deployment
Ready for Vercel/Netlify with default Vite settings.
